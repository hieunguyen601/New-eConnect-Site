# Fort Bend County eConnect Redesign

A modern, responsive redesign concept for the **Fort Bend County eConnect employee portal**. This project improves the website’s layout, navigation, accessibility, and mobile responsiveness while providing employees with easier access to county resources, departments, documents, and internal updates.

## Live Demo

[View the Live Website](https://hieunguyen601.github.io/New-eConnect-Site/)

## Project Overview

The original eConnect website contains many important employee resources, but its interface can be difficult to navigate. This redesign focuses on creating a cleaner and more organized user experience.

The website includes responsive navigation, department pages, employee resources, announcements, service awards, birthdays, and quick-access links.

> **Note:** This is a design and development project created for demonstration purposes. It is not the official Fort Bend County eConnect website.

## Run the Project Locally

1. Clone the repository:

```bash
git clone https://github.com/hieunguyen601/New-eConnect-Site.git
```

2. Open the project folder:

```bash
cd New-eConnect-Site
```

3. Open `index.html` in your browser.

You can also use the **Live Server** extension in Visual Studio Code:

1. Install Live Server.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

## Responsive Design

The layout is designed to adjust across different screen sizes.

On smaller screens:

* The desktop navigation changes into a hamburger menu.
* Navigation buttons move inside the mobile menu.
* Dropdown menus become easier to use by touch.
* Footer sections become expandable accordions.
* Content cards stack vertically.
* Text, spacing, and images resize for improved readability.

## Accessibility

The project includes several accessibility improvements:

* Semantic HTML elements
* Descriptive image alternative text
* ARIA labels
* `aria-expanded` states for dropdown controls
* Keyboard-friendly buttons
* Clear heading structure
* Improved color contrast
* External-link security attributes
